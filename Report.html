Site Report

A reflective discussion of my web development journey

Introduction & Learning Experience

This portfolio website represents my learning journey throughout this module where I explored the fundamentals of web development using HTML and CSS. At the beginning, understanding the structure of HTML documents and how CSS controls layout and styling felt overwhelming. However, through consistent practice and experimentation, I gradually became comfortable with semantic HTML, CSS Grid, and responsive design principles.

One of the biggest challenges I faced was debugging layout issues, especially when elements did not align as expected. Learning to use browser developer tools helped me identify problems such as incorrect grid placement, missing classes, or conflicting styles. Although frustrating at times, these challenges improved my problem-solving skills and patience.

Reflective Discussion of Module Experience

This module helped me understand not only how websites are built, but also why good design decisions matter. I learned that accessibility, readability, and responsiveness are just as important as visual appeal. The ups included successfully creating a responsive navigation bar and clean layouts using CSS Grid, while the downs involved trial-and-error with positioning and media queries.

Site Development & UI Design

Over the term, the site evolved from a simple static page into a structured multi-page portfolio. I focused on a consistent user interface across all pages, using the same header, navigation, and color theme. CSS Grid was used to organize content cleanly, making the site easier to scale and maintain.

Design Decisions

I chose a blue and white color palette to create a professional and calm visual experience. Blue represents trust and reliability, which aligns with my goal of presenting myself professionally. Sans-serif fonts were selected for better readability. Design inspiration was taken from modern portfolio websites such as Dribbble and Behance.
